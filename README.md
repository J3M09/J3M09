## About Me

I'm a developer who lives at the intersection of **software and hardware** — equally comfortable writing a PHP backend, designing a WinForms desktop app, flashing firmware onto an ESP32, and soldering the circuit it talks to.

Most developers stop at the screen. I go all the way to the hardware.

- 🔌 Building end-to-end IoT systems from embedded firmware to web dashboards
- 🖥️ Crafting desktop applications with C# (.NET / WinForms / MAUI)
- 🌐 Developing full-stack web apps with PHP, HTML/CSS/JS, and React
- 🤖 Exploring local/offline AI tooling for embedded and web workflows
- 🛠️ Always building something — just for the sake of making it work

|---|---|

| 🖥️ **Desktop** | WinForms / MAUI / WPF apps with clean architecture, async patterns, and real-time device integration |
| 🌐 **Web** | PHP + JS frontends with REST backends, Bootstrap UI, PDO database access, admin dashboards |
